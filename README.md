# YAAH
yaah is a simple AUR helper for Arch Linux written in Ruby Programming Language and uses pacman syntax.

## Requirements
`Ruby >= 2.4`
`curl`

## How to use?

### Search packages
`./archpkg -Ss packagename`

### Install packages (In Development)
`./archpkg -S packagename`
